Ejercicio 5:

Imagina que estamos creando un software para editar imágenes y necesitamos mostrar muchos píxeles en la pantalla. Cada píxel posee características como ubicación, color y nivel de opacidad.

En vez de generar un objeto individual para cada píxel, podemos emplear el patrón Flyweight para reciclar objetos ya creados, disminuyendo así el uso de memoria.
